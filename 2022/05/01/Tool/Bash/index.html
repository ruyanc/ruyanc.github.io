在 shell 脚本中，可以使用并行的方式来执行多个任务，以减少任务执行时间和提高系统资源利用率。通过并行执行任务，可以充分利用系统的 CPU 和 IO 资源，以提高任务的速度和效率。先定义待执行函数、参数和进程数：

下面是几种常用的并行方法

方法1

wait -n 命令会等待任意一个子进程结束并返回其状态。与 wait 命令不同的是，wait -n 命令可以让父进程在等待期间继续执行其他任务，而不必阻塞等待所有子进程执行完毕。

注意，bash版本需要高于4.3

方法3

使用 GNU parallel 工具来实现在 shell 中同时执行多个进程。

创建一个包含 30 个任务的文件 tasks.txt，每行一个任务。使用 parallel 命令来执行任务。使用 –jobs 参数指定使用 5 个进程来执行任务。使用 –joblog 参数指定记录任务执行的日志。
