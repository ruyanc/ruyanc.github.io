FasterHosts

FasterHosts 是个 Chrome 插件，主要原理是拦截浏览器的某些请求，将 domain 替换成访问速度较快的那个。

  • 加速访问 GitHub 相关的资源
  • 支持配置 Hosts
  1. 下载 FasterHosts 然后解压，找到 extension 子目录
  2. 打开 Chrome，输入: chrome://extensions/
  3. 打开「开发者模式」
  4. 选择「加载已解压的扩展程序」，然后定位到刚才解压的文件夹里面的 extension 目录，确定
  5. 这就安装好了，关闭「开发这模式」
