  • 删除 geo4palm_env.yml 文件中的以下字段：
  • 创建conda环境GEO4PALM
  • 激活GEO4PALM虚拟环境
  • install terracatalogueclient

  • 对应修改./JOBS/Wuhan/INPUT/config.static-Wuhan中的 [case] 和 [domain] 参数

  • 安装elevation - 基于集成的elevation工具包：https://github.com/bopen/elevation?tab=readme-ov-file
  • 测试安装
  • 选择区域范围

shp 建筑物数据准备

  • 网络资源：国内77个城市建筑物轮廓（带高度）数据分享 【环哥建筑物矢量shp】（测试可用）

https://www.bilibili.com/read/cv17043593/

  • 中国首套10米分辨率的建筑高度数据（tif格式）（尚未测试）

https://blog.csdn.net/weixin_63042008/article/details/130816753

https://zenodo.org/records/7827315

  • shp 文件可视化网站： https://mapshaper.org/

  • 移动数据到./JOBS/Wuhan/INPUT/shp文件夹下

  • 读取shp变量信息

  • shp 转换 tif
  • 对应修改./JOBS/Wuhan/INPUT/config.static-Wuhan

lu 土地利用类型

  • 注册 ESA WorldCover 账号

  • 修改./JOBS/Wuhan/INPUT/config.static-Wuhan

  • 把 esa_2020v1_lu.csv 放到./JOBS/Wuhan/INPUT/文件夹下

运行 GEO4PALM

  • 返回数据下载信息
  • 生成 Static Driver 数据到./JOBS/Wuhan/OUTPUT

n order for users to quickly check the quality of the dynamic driver generated by WRF4PALM, we provide a quick comparison script. Five variables are allowed (can be in uppercase or lowercase):

U - V - W - PT - QV

Three plot types are provided:

zcross: vertical cross sections of west/east/south/north boundaries for the user specified variable and timestamp

then the script will ask for the timestamp:

Once the timestamp is given, the script will return a comparison plot.

pr: vertical profiless of west/east/south/north boundaries for the user specified variable and timestamp

then the script will ask for the timestamp:

Once the timestamp is given, the script will return a comparison plot.
Note that the vertical profiles are horizontally averaged and hence the comparison only gives a approximate reference regarding the performance of WRF4PALM.

ts: time series of west/east/south/north boundaries for the user specified variable and altitude

then the script will ask for the altitude:

Once the vertical level is given, the script will return a comparison plot.
Note that the time series are horizontally averaged and hence the comparison only gives a approximate reference regarding the performance of WRF4PALM.
