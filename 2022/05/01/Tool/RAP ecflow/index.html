https://perillaroc.github.io/ecflow-tutorial-cn/chap02/use-ecflow-ui/

  • 选择 Servers 菜单 → Manage Servers → Add Server：

    为 ecflow 服务器取个名字（rap），添加 Host （localhost）和 Port（3141），点击添加按钮

  • 选择 Tools 菜单 → Preferences：

    左边栏选择 Menus → 将右侧 menu mode 修改为 Administrator

  • 从 Servers 菜单的 All servers… 中选择要显示的服务

    层层点击小三角，即可分层显示作业目录
