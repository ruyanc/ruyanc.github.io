matplotlib中colormap的常用设置

常用指定颜色的方法

  1. RGB或者RGBA(red, green, blue, alpha)数组，浮点数[0, 1]，如 (0.1, 0.2, 0.5) 或 (0.1, 0.2, 0.5, 0.3)

  2. 不区分大小写的hex RGB or RGBA，如’#0f0f0f’或’#0f0f0f80’

  3. 灰度值，[0,1]中的浮动数的字符串表示形式，’0’表示黑色，’1’表示白色，’0.8’表示灰色

  4. 一些基本颜色的单字符简写。

    • • b：蓝色
    • • g：绿色
    • • r：红色
    • • c：青色
    • • m：红色
    • • y：黄色
    • • k：黑色
    • • w：白色
    1. 具体颜色名称，如’red’,’blue’
