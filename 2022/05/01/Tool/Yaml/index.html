1. Yaml

  • Yaml 基本语法：

    • 大小写敏感

    • 使用缩进，表示层级关系

    • 缩进不允许使用tab，只允许空格

2. gprof使用

查看用户自定函数的调用关系及运行时间。编译一般使用 -O3 。查看：参考链接

  • 基本原理：

    • gcc/g++ 编译时加上 -pg

    • 编译器在每个函数的开头加 mcount 函数调用

    • 保存函数的调用关系图和函数的调用信息

    • 最终在程序退出时保存在 gmon.out 文件中

  • 基本使用方法：

    1. 使用 -pg 参数编译程序

    2. 运行程序，并正常退出

    3. 使用 gprof 命令查看 gmon.out 文件

  • 使用举例

    测试运行 Test_IOGrapes.exe

    1. cmake时，添加参数-pg: SET (CMAKE_Fortran_FLAGS "${CMAKE_Fortran_FLAGS} -pg")

    2. 编译并运行 Test_IOGrapes.exe，运行成功生成 gmon.out

    3. 使用 gprof查看 gmon.out：gprof Debug/ Test_IOGrapes.exe gmon.out > gprof.txt

    4. 生成运行及调用信息图：gprof Debug/Test_IOGrapes.exe gmon.out | gprof2dot |dot -Tpng -o example.png
