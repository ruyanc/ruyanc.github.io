MAKE_DEPOSITORY_default

存放个人代码的修改

源代码 + 编译中间文件 + 可执行文件存放位置

可执行文件：palm 为palm主程序 + combine_plot_fields.x 为合并各节点的输出文件打到一个文件中

.palm.config.default

修改 %source_path为/work/home/acyii6v8kt/WORK/Palm/palm_model_system-v23.10/MAKE_DEPOSITORY_default 以保证原代码的整洁

%compiler_options 添加 -g -traceback 进行代码调试，给出具体出错信息

./bin/palmbuild

首次安装好后重新编译时使用，即使用.palm.config.default配置文件

新建 test 文件夹用于存在运行个例

  • 准备输入文件和namelist:

PID_STATIC 为nc格式的静态输入文件，包括网格信息，网格位置

PIDS_DYNAMIC 为nc格式的动态输入文件，包括边界条件（uvtpq 六个面，一般设置侧向，上下为默认）、初始条件

PARIN 为namelist

ENVPAR 为计算节点设置
