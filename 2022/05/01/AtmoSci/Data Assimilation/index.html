What is data assimilation?

Data assimilation is the process of estimating the state of a dynamical system by combing observational data with a priori estimate of the state (often from a numerical model forecast)

Why not just use the observations?

  • We may only observe part of the state

  • We may observe a nonlinear function of the state, e.g. satellite radiances

观测算子：从模式变量到观测变量 （可能非常复杂）

  • We need to allow for uncertainties in the observations (and in the a prior estimate)

观测和模式均存在误差

Unbiased: 误差的平均值为0

Uncorrelated: 观测误差和模式误差不相关

两个未知变量，需要两个条件：

当观测和模式的方差相等时，分析场等于二分之一的观测+二分之一的模式

当观测（模式）方差较大时，模式（观测）的权重较大；

p(x|y) 当有观测y时模式x的最佳估算

more weight for the observation because of less uncertainty
