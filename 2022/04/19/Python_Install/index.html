conda -h

基于python3.8版本创建一个名字为python38的环境

conda create --name python38 python=3.8

激活此环境

activate python38

再来检查python版本，显示是 3.8

python -V

退出当前环境

deactivate python38

删除该环境

conda remove -n python38 --all
